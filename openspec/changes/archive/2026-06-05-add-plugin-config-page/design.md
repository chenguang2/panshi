## Key Decisions

| 决策 | 选择 | 理由 |
|---|---|---|
| 布局 | CSS Grid 卡片网格 | 设计稿为卡片布局 |
| 表单 | Ant Design `<a-modal>` + `<a-form>` | 与上游/路由管理一致 |
| 操作按钮 | 直接显示在卡片上（非下拉菜单） | 与集群管理中插件组一致 |
| 数据复用 | `useClusterPluginConfigs` composable | 复用现有逻辑 |
| 跨集群列表 | 新增 `GET /api/v1/plugin_configs` | 参考 upstreams/routes 实现 |
| 集群筛选 | "全部集群"为默认（非"选择集群"） | 按需求 |
| 查看详情 | Drawer 展示 | 与集群管理一致 |
| 删除 | `showDeleteConfirm` + `executeDeleteWithProgress` | 与上游管理一致 |
| 发布 | `PublishConfirmModal` + `executePublish` | 与上游管理一致 |
