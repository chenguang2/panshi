## 设计方案

提供 3 种 UI 方案对比。

---

## 方案 A：卡片分组 + 内联展开

### 入口：节点操作列新增按钮

```
┌──────────────────────────────────────────────┐
│  10.0.1.1:9180       ✅ 在线                │
│  ┌─────────┐ ┌──────────────┐               │
│  │ 编辑    │ │ 数据库对比   │               │
│  └─────────┘ └──────────────┘               │
└──────────────────────────────────────────────┘
```

### 对比结果视图（在当前节点下方展开）

```
┌─────────────────────────────────────────────────────────────┐
│ 🔍 配置对比: 10.0.1.1:9180                  ┌──────────┐   │
│ 上次同步: 2026-05-14 10:30                  │ 重新对比  │   │
│                                              └──────────┘   │
│                                                              │
│ ┌─ 上游服务 (2) ─────────────────────────────────────────┐  │
│ │  ✅ api-v1                    DB: 10.0.1.1:8080         │  │
│ │                              Edge: 10.0.1.1:8080        │  │
│ │  ❌ user-service             DB: 10.0.1.2:8080      ←  │  │
│ │    ┌ 点击展开差异 ─────────────────────────────────┐   │  │
│ │    │  字段       │ 数据库               │ Edge 节点  │   │  │
│ │    │─────────────┼──────────────────────┼────────────│   │  │
│ │    │ target      │ 10.0.1.2:8080        │ 10.0.1.3:8080│  │  │
│ │    │ weight      │ 100                  │ 50          │  │  │
│ │    └─────────────────────────────────────────────────┘   │  │
│ │  ➕ monitor                 仅 DB 有                    │  │
│ └──────────────────────────────────────────────────────────┘  │
│                                                              │
│ ┌─ 路由规则 (3) ─────────────────────────────────────────┐  │
│ │  ✅ /api/users              一致                        │  │
│ │  ❌ /api/orders             权重不同                    │  │
│ │  ➖ /api/old                 仅 Edge 有                 │  │
│ └──────────────────────────────────────────────────────────┘  │
│                                                              │
│ ┌─ 插件组 (2) ───────────────────────────────────────────┐  │
│ │  ✅ log-plugin              一致                        │  │
│ │  ❌ rate-limit              DB 有配置，Edge 上缺失      │  │
│ └──────────────────────────────────────────────────────────┘  │
│                                                              │
│ ┌─ 全局规则 (1) ─────────────────────────────────────────┐  │
│ │  ❌ monitor-rule            DB 配置不同                  │  │
│ └──────────────────────────────────────────────────────────┘  │
│                                                              │
│ ┌─ 插件元数据 (2) ───────────────────────────────────────┐  │
│ │  ✅ log_process             一致                        │  │
│ │  ➕ data_center             仅 DB 有                    │  │
│ └──────────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
```

**特点**：
- 结果直接在节点卡片下方展示，不弹窗
- 每个资源组是折叠卡片，可以展开/收起
- 状态图标：✅ 一致、❌ 不一致、➕ DB有Edge无、➖ Edge有DB无
- 不一致的项高亮，点击展开字段级对比表格
- 简单直观，适合少量数据

## 方案 B：独立页面 + 双栏对比视图

### 入口：点击"数据库对比"跳转到独立页面

```
┌─────────────────────────────────────────────────────────────┐
│ 磐石 Admin  >  集群管理  >  配置对比                        │
│                                                             │
│ 集群: production-cluster  │  节点: 10.0.1.1:9180  [切换▼]  │
│                                                             │
│ 统计: 上游 5 | 路由 12 | 插件组 3 | 全局规则 2 | ...       │
│       ✅ 一致 15  │  ❌ 不一致 4  │  ➕ 仅DB 2  │  ➖ 仅Edge 1 │
│                                                             │
│ ┌───────────┬───────────────────┬─────────────────────────┐ │
│ │ 资源类型  │ 数据库配置        │ Edge 配置               │ │
│ ├───────────┼───────────────────┼─────────────────────────┤ │
│ │           │                   │                         │ │
│ │ 上游服务  │  api-v1           │  api-v1                 │ │
│ │           │  target:10.0.1.1  │  target:10.0.1.1       │ │
│ │           │  weight:100       │  weight:100             │ │
│ │           ├───────────────────┼─────────────────────────┤ │
│ │           │  ⚠️ user-service  │  ⚠️ user-service       │ │
│ │           │  target:10.0.1.2  │  target:10.0.1.3  🔴   │ │
│ │           │  weight:100       │  weight:50         🔴   │ │
│ │           ├───────────────────┼─────────────────────────┤ │
│ │           │  monitor    ➕     │                         │ │
│ ├───────────┼───────────────────┼─────────────────────────┤ │
│ │ 路由规则  │  ...              │  ...                    │ │
│ └───────────┴───────────────────┴─────────────────────────┘ │
```

**特点**：
- 独立全屏页面，对比信息更丰富
- 双栏并排对比，一目了然
- 支持切换节点
- 适合数据量大、需要仔细比对的场景

## 方案 C：侧边栏抽屉 + 分组比较器

### 入口：同方案A按钮，点击打开右侧抽屉

```
┌──────────────────────────────────────────────────┐
│  配置对比: 10.0.1.1:9180                    [✕] │
│──────────────────────────────────────────────────│
│                                                  │
│  概览                                            │
│  ┌──────────────────────────────────────┐        │
│  │  一致 12 │ 差异 3 │ 仅DB 2 │ 仅Edge 1│        │
│  └──────────────────────────────────────┘        │
│                                                  │
│  ▼ 上游服务 (3)              2 不一致  ➕ 1      │
│  ┌──────────────────────────────────────┐        │
│  │  □ api-v1         目标: 一致        │        │
│  │  □ user-service   目标: 10.0.1.2 ❌  │        │
│  │  │  → Edge: 10.0.1.3               │        │
│  │  │  → DB:   10.0.1.2               │        │
│  │  □ monitor        仅在数据库       │        │
│  └──────────────────────────────────────┘        │
│                                                  │
│  ▼ 路由规则 (5)              1 不一致            │
│  ...                                             │
│                                                  │
│  ▼ 插件组 (2)              1 不一致              │
│  ...                                             │
│                                                  │
│  ▼ 全局规则 (1)              1 不一致            │
│  ...                                             │
│                                                  │
│  ▼ 插件元数据 (2)              全部一致          │
│  ...                                             │
│                                                  │
└──────────────────────────────────────────────────┘
```

**特点**：
- 侧边抽屉，不离开当前页面
- 分组折叠 + 差异项直接在行内展开
- 紧凑，适合频繁操作
- 概览统计条快速了解整体情况

---

## 三种方案对比

| 维度 | 方案A（内联展开） | 方案B（独立页面） | 方案C（侧边抽屉） |
|------|:---------:|:---------:|:---------:|
| 实现复杂度 | 低 | 高 | 中 |
| 数据量适应性 | 中小 | 大 | 中 |
| 不离开当前页面 | ✅ | ❌ | ✅ |
| 对比直观度 | 中（逐项展开） | 高（双栏并排） | 中（行内标注） |
| 字段级对比 | 点击展开 | 始终可见 | 点击展开 |
| 节点切换 | ❌ 需重新点击 | ✅ | ❌ |
| 移动端友好 | ✅ | ❌ | ❌ |

**推荐方案 A**：实现简单，用户直接在节点下方查看差异，符合"不用弹窗"的要求。差异项点击展开字段对比，不占空间。
