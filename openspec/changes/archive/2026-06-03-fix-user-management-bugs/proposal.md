## Why

用户管理页面存在多处 UI 和文案问题，影响使用体验：
1. 用户列表表格中角色选择下拉和启用标识列宽度过大，与设计稿不符
2. 权限列显示占位文本"在编辑中设置"而非实际的权限标签，用户无法直观看到权限
3. 编辑页面的权限列表中"边缘节点管理"与侧边栏菜单名"Edge直连"不一致
4. 编辑页面的重置密码输入框过宽，导致布局被挤到下一行

## What Changes

1. **用户列表 - 列宽优化**：为角色列和状态列设置合适的固定宽度，按设计稿样式调整
2. **用户列表 - 权限列显示标签**：不再显示"在编辑中设置"，改为显示用户的权限标签（如"路由管理"、"上游管理"等小标签），管理员显示"全部权限"
3. **编辑页面 - 权限标签改名**：将"边缘节点管理"改为"Edge直连"，与侧边栏菜单保持统一
4. **编辑页面 - 重置密码输入框宽度**：限制重置密码输入框宽度，使输入框和按钮在同一行

## Capabilities

### New Capabilities
- `user-management-ui`: 用户管理页面的 UI 展示能力，包括用户列表的列渲染、权限标签展示、编辑表单布局

（无现有 spec 需要修改）
