dns_upstream插件，如果启用健康检查，支持4层和7层主动检查和被动检查


```json
"dns_upstream": {  
    "hosts": {  
        "test.local": {  
            "checks": {"type": "http", "active": {}, "passive": {}},  
            "ttl_valid": 10,  
            "type": "chash",  
            "nodes": {  
                "127.0.0.1:9981": [],  
                "192.161.1.1:9980": [],  
                "192.18.1.2": []  
            }  
        }  
    }  
},  
```

加了个ttl_valid可以配置，默认10s，健康检查里，
加了个"type"配置，默认检查类型7层是http,4层tcp,
dns_upstream是在四层用的，检查默认按tcp检查，不检查http,
