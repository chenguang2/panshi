# JSON 对比工具设计

## 处理流程

```
JSON A → parse → sortKeys → JSON.stringify(, null, 2) → 分行
JSON B → parse → sortKeys → JSON.stringify(, null, 2) → 分行
                    ↓
              逐行对比（取最大行数，缺行补空）
                    ↓
          v-html 渲染到下方结果框
```

## 着色规则

| 状态 | 颜色 |
|---|---|
| 相同行 | 灰色  |
| A有B无（删除） | 红底红字 `- 内容`，删除线 |
| B有A无（新增） | 绿底绿字 `+ 内容` |
| A≠B（修改） | 先红行（A旧值）+ 绿行（B新值）成对输出 |

## UI 布局

- 上方：双栏输入框，不变
- 下方：固定高度（max-height: 400px）、可滚动的结果框，等宽字体
- 结果框输出两边完整 JSON，行前带 ` `（相同）、`-`（删除）、`+`（新增）前缀
