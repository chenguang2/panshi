# Edge Admin 功能对照清单

> ✅ = 磐石 Admin 已实现
> ⬜ = 未实现
> ➖ = 不适用 / 属于运维部署范畴

---

## 一、应用 (Applications)

### HTTP 应用

- [ ] **创建应用** — Edge Admin 以"应用"为管理单元，磐石以"路由"为单元，缺少应用层抽象
- [x] **发布应用的修改** — 已有发布到 Edge 节点的完整流程
- [ ] **应用设置** — 应用级别的通用设置（域名绑定、SSL 等）
- [x] **反向代理到上游** — 已有上游管理 + 路由关联上游
- [x] **代理中上游节点的重试次数** — 已有 retries 字段
- [ ] **代理缓存** — ❌ 缺失，路由级缓存策略配置
- [ ] **gRPC 代理** — ❌ 缺失
- [ ] **绑定上游到 k8s 服务** — ❌ 缺失
- [ ] **启用 HTTP/3** — ❌ 缺失
- [ ] **CC 攻击日志** — ❌ 缺失
- [ ] **响应体压缩**
  - [ ] Gzip 压缩 — ❌ 缺失
  - [ ] Brotli 压缩 — ❌ 缺失
  - [ ] Zstandard（zstd）压缩 — ❌ 缺失

### 其他应用类型

- [ ] **SSL 应用 (SSL/TLS 卸载)** — ❌ 缺失，无证书管理
- [ ] **TCP 应用** — ❌ 缺失，只支持 HTTP
- [ ] **HTTP 正向代理** — ❌ 缺失
- [ ] **Socks5** — ❌ 缺失
